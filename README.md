## Prerequisites

- Node.js (v14 or v16 recommended)
- npm (v6 or higher)

## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/Madhugoud9949/rock_paper_scissors.git
    cd rock_paper_scissors
    ```

2. Install dependencies:

    ```bash
    npm install
    ```

## Running the App

To start the development server:

npm start
